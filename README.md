## 🛠️ First-Time Setup

When you open Agent for the first time, set it up in a way that fits your study style.

- Create a list for each job role you want to prepare for

## 📌 Basic Use

### Add a question

Type a question you want to practice. Keep the text short and clear.

Example:
- Tell me about yourself
- Why do you want this job?
- How do you handle pressure?

### Add your answer

Write a short answer below each question. Use simple words and keep your points easy to scan.

### Group by topic

You can sort notes into groups like:

- Self-introduction
- Work history
- Technical skills
- Problem solving
- Teamwork
- Company fit

### Review before interviews

Use Agent as a quick review tool before a call or in-person interview. Read each question and answer once or twice, then focus on the items you still miss.
